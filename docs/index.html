Clipboard, sync LAN và gửi file

OpenCB giúp clipboard đi cùng bạn.

Lưu lịch sử clipboard lớn, tìm lại nhanh, ghim và gắn thẻ nội dung, đồng bộ giữa Windows và Android trong cùng mạng LAN, kèm gửi file trực tiếp như một công cụ nội bộ gọn nhẹ.

Lịch sử clipboard
LAN
Link tài liệu dự án URL được nhận diện riêng, mở lại nhanh khi cần.
Ảnh vừa copy từ trình duyệt Xem preview, ghim lại hoặc gửi sang thiết bị khác.
Gửi file tới điện thoại Chọn nhiều file/folder và gửi trong cùng mạng nội bộ.
01

Clipboard có tổ chức

Lưu văn bản, URL, ảnh, path file; tìm kiếm nhanh, ghim mục quan trọng và gắn thẻ theo cách làm việc của bạn.

02

Sync LAN riêng tư

Các thiết bị đã ghép nối tự thấy nhau trong cùng Wi-Fi/VPN, đồng bộ clipboard mới mà không cần cloud.

03

Gửi file nội bộ

Gửi file hoặc folder giữa Windows và Android, có xác nhận nhận file, tiến trình truyền và lịch sử hoạt động.
